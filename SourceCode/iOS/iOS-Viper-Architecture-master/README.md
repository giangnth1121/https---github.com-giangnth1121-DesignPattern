## Just fork, clone, build, run and learn VIPER architecture for IOS.

```sh
git clone https://github.com/MindorksOpenSource/iOS-Viper-Architecture.git
cd iOS-Viper-Architecture
pod install
open iOS-Viper-Architecture.xcworkspace
```
